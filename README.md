## What is?

> ### A terminal with unix features, for a better experience on Windows 
>
> #### :construction: This project is not yet completed, but feel free to make your contribution

## How to run?

- **npm**

```bash
npm i

npm run dev
```

- **yarn**

```bash
yarn

yarn dev
```
